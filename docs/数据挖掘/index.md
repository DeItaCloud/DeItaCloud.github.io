# 数据挖掘

## 数据挖掘任务

从海量数据中用机器学习、深度学习、大数据的方法提取数据价值

## 数据挖掘目标

发现数据价值，提供业务决策帮助或直接提供决策

## 数据挖掘对象

结构化数据
