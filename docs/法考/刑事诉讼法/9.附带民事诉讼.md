# 附带民事诉讼✨✨✨

以刑事诉讼为前提

## 赔偿范围

因被告人的犯罪行为造成的 **物质的**、**实际的**、**必然的** 损失

!!! note ""
        不支持精神损害赔偿（打残赔轮椅、打死赔棺材）、偷抢损失

        能赔：   
        医护交付减收入

## 当事人

凡是因为犯罪行为造成损失的都可以作为原告人

!!! note "赔偿人"    
        被告人、被告人共同犯罪人、他们的遗产继承人、他们的监护人

!!! note ""
        共同侵害人原告有选择权，但是同案犯在逃不能告

## 期间

立案后

## 保全

财产保全等

诉前保全担保必须，诉中可以

案件受理后15天内不起诉保全会被解除

## 附带民诉审判程序

1. 应当调解
2. 调解书强制力和判决相同
3. 履行完毕的调解书，调解书不可上诉

原告不来撤诉 被告不来可以缺席裁判还可以拘传（刑事被告人）
