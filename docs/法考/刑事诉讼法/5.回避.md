# 回避✨✨

侦检审书翻鉴可回避

当法诉辩可申请

有影响公正的关系都回避
!!! note ""
        专门知识的人参照鉴定人

        证人不可替代不需要回避

!!! note ""
        法院不合适适用指定管辖（管辖不宜）

## 参前不参后

侦查不能参加审查起诉

!!! note ""
        自行补充侦查的人除外

二审发回一审，一审合议庭需要回避

再回到二审，二审合议庭不需要回避（二审法院没有实质参加审理）

## 回避决定

一般找老大 老大找组织（公安老大-检委会）

书翻鉴谁聘请谁决定

被回避人行为效力由老大决定

!!! note "暂停工作"
                被申请回避就要暂停工作，决定后立即生效，复议不影响决定效力   

                一般要暂停侦查才例外
