Matrix Factorization¶

Matrix factorization is a class of collaborative filtering algorithms used in recommender systems. Matrix factorization algorithms work by decomposing the user-item interaction matrix into the product of two lower dimensionality rectangular matrices.

Assuming that user $u$ and item $i$ are mapped to factor vectors $P_u$ and $Q_i$ respectively, we estimate the rating of item $i$ given by user $u$ as: $$ \hat R_{ui} = P_u^T Q_i. $$ Thus, to learn the factor vectors, the system can minimize the regularized squared error on the set of known ratings: $$ \min_{P,Q}\frac{1}{2}\sum_{(u,i)\in \mathcal{K}}(R_{ui}-P_u^TQ_i)^2 + \frac{\lambda}{2}(||P||^2 + ||Q||^2) $$ Here, $\mathcal{K}$ is the set of the ($u$,$i$) pairs for which $R_{ui}$ is known (the training set).

Probilistic Matrix Factorization¶

We define the conditional distribution over the observed ratings as: $$ p(R|P,Q,\sigma^2)=\prod_{u=1}^N\prod_{i=1}^M[\mathcal{N}(R_{ui}|P_u^TQ_i, \sigma^2)]^{I_{ui}}, $$ where $\mathcal{N}(x|\mu, \sigma^2)$ is the probability density function of the Gaussian distribution with mean $\mu$ and variance $\sigma^2$, and $I_{ui}$ is the indicator function that is equal to 1 if user $u$ rated item $i$ and equal to 0 otherwise. We also place zero-mean spherical Gaussian priors on user and item feature vectors: $$ p(P|\sigma_P^2) = \prod_{u=1}^N\mathcal{N}(P_u|0,\sigma_P^2\mathbf{I}),\quad p(Q|\sigma_Q^2) = \prod_{i=1}^M\mathcal{N}(Q_i|0,\sigma_Q^2\mathbf{I}). $$ To begin with, the log of $\mathcal{N}(x|\mu, \sigma^2)$ is: \begin{align} \ln \mathcal{N}(x|\mu, \sigma^2) = & \ln \frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma ^2}}\\ = & - \frac{1}{2}\ln2\pi -\frac{1}{2}\ln \sigma^2 - \frac{(x-\mu)^2}{2\sigma ^2} \end{align} Then the log of the posterior distribution over the user and movie features is: \begin{align} \ln p(P,Q|R,\sigma^2,\sigma_P^2,\sigma_Q^2) = &\ln\frac{p(R|P,Q,\sigma^2)p(P|\sigma_P^2)p(Q|\sigma_Q^2)}{p(R|\sigma^2,\sigma_P^2,\sigma_Q^2)}\\ = &-\frac{1}{2\sigma^2}\sum_{u=1}^N\sum_{i=1}^MI_{ui}(R_{ui}-P_u^TQ_i)^2 - \frac{1}{2\sigma_P^2}\sum_{u=1}^NP_u^TP_u - \frac{1}{2\sigma_Q^2}\sum_{i=1}^MQ_i^TQ_i\\&- \frac{1}{2}((\sum_{u=1}^N\sum_{i=1}^MI_{ui})\ln\sigma^2 + ND\ln\sigma_P^2 + MD\ln\sigma_Q^2) + C, \end{align} where $C$ is a constant that does not depend on the parameters. Maximizing the log-posterior over user and item features with hyperparameters (i.e. the observation noise variance and prior variances) kept fixed is equivalent to minimizing the sum-of-squared-errors objective function with quadratic regularization terms: \begin{align} E &= \frac{1}{2}\sum_{u=1}^N\sum_{i=1}^MI_{ui}(R_{ui}-P_u^TQ_i)^2 + \frac{\lambda_P}{2}\sum_{u=1}^N ||P_u||^2 + \frac{\lambda_Q}{2}\sum_{i=1}^N ||Q_i||^2\\ &= \frac{1}{2}\sum_{u=1}^N\sum_{i=1}^MI_{ui}(R_{ui}-P_u^TQ_i)^2 + \frac{\lambda_P}{2}||P||^2 + \frac{\lambda_Q}{2}||Q||^2, \end{align} where $\lambda_P=\sigma^2/\sigma_P^2$ and $\lambda_Q=\sigma^2/\sigma_Q^2$.

import tensorflow as tf
import numpy as np

# hyperparameters
learning_rate = 1e-2
user_l2_factor = 1e1
item_l2_factor = 1e1
max_training_step = 1000
train_rate = 0.8
embedding_size = 8 # 每个用户的向量纬度

rating_file_path = '/kaggle/input/movielens_10k'
data = np.loadtxt(fname=rating_file_path, delimiter='\t')

# calculate the numbers of users and items
user_set = set()
item_set = set()
for i, j, k in data:
    user_set.add(int(i))
    item_set.add(int(j))
user_num = len(user_set)
item_num = len(item_set)

# split data
np.random.shuffle(data)
t = int(len(data)*train_rate)
dtrain = data[:t]
dtest = data[t:]

# construct graph
user_embeddings = tf.Variable(tf.truncated_normal([user_num, embedding_size], mean=0, stddev=0.01))
item_embeddings = tf.Variable(tf.truncated_normal([item_num, embedding_size], mean=0, stddev=0.01))
# user_bias = tf.Variable(tf.zeros([user_num], tf.float32))
# item_bias = tf.Variable(tf.zeros([item_num], tf.float32))
# rating_bias = tf.Variable(0.0)

user_id = tf.placeholder(tf.int32, shape=[None])
item_id = tf.placeholder(tf.int32, shape=[None])
y = tf.placeholder(tf.float32, shape=[None])

user_emb = tf.nn.embedding_lookup(user_embeddings, user_id)
item_emb = tf.nn.embedding_lookup(item_embeddings, item_id)
# ubias = tf.nn.embedding_lookup(user_bias, user_id)
# ibias = tf.nn.embedding_lookup(item_bias, item_id)

y_pre = tf.reduce_sum(user_emb * item_emb, 1)# + ubias + ibias + rating_bias

loss = 0.5 * tf.reduce_sum(tf.square(y - y_pre)) + \
       0.5 * user_l2_factor * tf.reduce_sum(tf.square(user_embeddings)) + \
       0.5 * item_l2_factor * tf.reduce_sum(tf.square(item_embeddings))

train_step = tf.train.AdamOptimizer(learning_rate).minimize(loss)
rmse = tf.sqrt(tf.reduce_mean(tf.square(y - y_pre)))

with tf.Session() as sess:
    sess.run(tf.global_variables_initializer())
    for i in range(max_training_step):
        feed_dict = {user_id: dtrain[:, 0],
                     item_id: dtrain[:, 1],
                     y: np.float32(dtrain[:, 2])}
        sess.run(train_step, feed_dict)

        rmse_train, loss_value = sess.run([rmse, loss], feed_dict={user_id: dtrain[:, 0], item_id: dtrain[:, 1], y: np.float32(dtrain[:, 2])})
        rmse_test = sess.run(rmse, feed_dict={user_id: dtest[:, 0], item_id: dtest[:, 1], y: np.float32(dtest[:, 2])})
        print('----round%4d: rmse_train: %f, rmse_test: %f, loss: %f' % (i + 1, rmse_train, rmse_test, loss_value))

Adding Biases¶

Till now we model the rating as: $$ \hat R_{ui} = P_u^T Q_i. $$ The equation tries to capture the interactions between users and items that produce the different rating values. However, much of the observed variation in rating values is due to effects associated with either users or items, known as biases, independent of any interactions. Adding the biases into current model, we have: $$ \hat R_{ui} = P_u^T Q_i + b_u + b_i + \mu_r. $$
